快速入门

# 安装Sass
npm install sass --save-dev

# 创建Sass文件
echo '// 变量
$primary-color: #CC6699;
$font-stack: "Helvetica Neue", Arial, sans-serif;

// 混入
@mixin flex-center {
    display: flex;
    justify-content: center;
    align-items: center;
}

// 嵌套
.container {
    width: 100%;
    max-width: 1200px;
    margin: 0 auto;
    
    .header {
        background: $primary-color;
        padding: 20px;
        
        h1 {
            font-family: $font-stack;
            color: white;
        }
    }
    
    .content {
        @include flex-center;
        padding: 40px;
    }
}

// 继承
%button-base {
    padding: 10px 20px;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

.button-primary {
    @extend %button-base;
    background: $primary-color;
    color: white;
}

.button-secondary {
    @extend %button-base;
    background: white;
    color: $primary-color;
}' > styles.scss

# 编译Sass
npx sass styles.scss styles.css

# 监听变化
npx sass --watch styles.scss:styles.css

配置示例

// Webpack配置
// webpack.config.js
module.exports = {
    module: {
        rules: [
            {
                test: /\.scss$/,
                use: [
                    "style-loader",
                    "css-loader",
                    {
                        loader: "sass-loader",
                        options: {
                            // 使用Dart Sass
                            implementation: require("sass"),
                            // 配置选项
                            sassOptions: {
                                outputStyle: "compressed"
                            }
                        }
                    }
                ]
            }
        ]
    }
};

// Vite配置
// vite.config.js
import { defineConfig } from "vite";

export default defineConfig({
    css: {
        preprocessorOptions: {
            scss: {
                // 全局变量
                additionalData: `
                    $primary-color: #CC6699;
                    $secondary-color: #6699CC;
                `
            }
        }
    }
});

与其他CSS预处理器对比

📊

vs Less

Sass功能更强大，社区更活跃。Less更简单，学习曲线低，适合简单项目。

功能强大社区简单
📊

vs Stylus

Sass语法更规范，工具支持更好。Stylus语法更灵活，但社区较小。

语法规范工具支持灵活性
📊

vs PostCSS

Sass是预处理器，扩展CSS语法。PostCSS是后处理器，转换CSS代码。可配合使用。

预处理器后处理器配合使用
